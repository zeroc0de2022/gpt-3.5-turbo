Chat использует модель Искусственного Интеллекта gpt-3.5-turbo от OpenAI

Создана на основе статьи https://metanit.com/python/recipes/5.1.php

Для корректной работы необходимы следующие условия:
1. Создать API ключ доступа по адресу https://platform.openai.com/account/api-keys
2. Иметь установленные Python, библиотеки openai и pyqt5. Если Python или библиотеки не установлены, воспользуетесь файлами bat из папки install для установки необходимых компонентов.
3. Для запуска используется файл run.bat

В чате предусмотрены следующие настройки:
- Хранить историю переписки, сохраняется в папке history

- Учет переписки в ответе
- - Не учитывать в ответе историю переписки: = модель не будет знать ничего о предыдущих сообщениях и не будет учитывать их в своем ответе. Подходит для одноразовых вопросов, которые не требуют продолжения диалога. 
- - Учитывать в ответе всю историю переписки: = модель будет помнить в диалоге всю историю сообщений за текущий день, эта опция токенозатратна и рекомендуется к использованию при сложных диалогах, требующих от модели помнить всю историю сообщений.
- - Учитывать в ответе последние 5 сообщений: = модель помнит информацию о 5 последних сообщениях
- - Учитывать в ответе последние 3 сообщений: = модель помнит информацию о 3 последних сообщениях

- Количество токенов: = количество токенов по умолчанию 50, максимум можно задать 2016. Чем больше токенов, тем объемнее ответ от модели, и тем быстрее затрачивается лимит 4096. Начните с 50, и увеличивается количество, пока ответы ИИ не будут удовлетворительными.

- API ключ: - необходимо получить по адресу https://platform.openai.com/account/api-keys
